# Standard Core by The Story

## Environment variables

Provide this variables in `.env` file of the project not this package.

- **NEXT_PUBLIC_DEFAULT_LOCALE** - default locale, "en" by default
- **NEXT_PUBLIC_LOCALES** - available locales separated by comma, "en,pl" by default
- **NEXT_PUBLIC_LOCALE_PREFIX** - locale prefix, "always" by default, see [next-intl docs](https://next-intl.dev/docs/routing#locale-prefix)
